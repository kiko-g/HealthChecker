## Requirements

To run this app without using Docker, we recommend using node v16.11.0 and npm 8.0.0

## How to run

Using node >= 16 and 2 separate shells:

```shell
cd backend
npm install
npm run start
```

```shell
cd frontend
npm install
npm run start
```

Using docker compose:

```shell
docker compose up
```
